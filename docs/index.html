Brink contract Python package index

This is the PEP 503 simple-index for the brink-contract Python package. The index itself is publicly readable. The wheels it points at live as release assets on the private getbrink/brink repo and require a GitHub PAT to download.

Index URL

https://getbrink.github.io/contract-index/simple/

Install

Configure ~/.netrc with a GitHub PAT scoped to getbrink/brink Contents: Read:

machine github.com
  login x-token
  password $BRINK_PYPI_TOKEN

Then:

pip install \
    --index-url https://getbrink.github.io/contract-index/simple/ \
    --extra-index-url https://pypi.org/simple/ \
    brink-contract

For uv-native projects

[project]
dependencies = ["brink-contract>=0.1.2,<0.2.0"]

[[tool.uv.index]]
name = "getbrink"
url = "https://getbrink.github.io/contract-index/simple/"
explicit = true

[tool.uv.sources]
brink-contract = { index = "getbrink" }
